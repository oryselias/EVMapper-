# EVMapper

Application to map EV charging stations efficiently.

## Project Structure

- `backend-spring/` — Spring Boot backend (Java 17+, Maven, PostgreSQL + PostGIS)
- `ev-mapper-ui/` — React/TypeScript frontend

## Prerequisites

- Java 17 (LTS) or newer
- Maven 3.6+
- PostgreSQL with PostGIS extension
- Node.js 18+ (for frontend)

- **Backend:** Spring Boot 3.x, PostgreSQL, PostGIS
- **Frontend:** React, TypeScript
- **Maps:** Leaflet / OpenStreetMap
